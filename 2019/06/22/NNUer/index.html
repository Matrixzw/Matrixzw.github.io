南师食堂点评

发表于 2019-06-22 阅读次数：

1.1 首先一楼的小熊厨房可是4年+的老店了，里面的东西的话没得说，就是感觉扒饭，饭有点少，吃不太饱。我喜欢他家的酸汤鱼，酸菜鱼，11r的。可以免费加饭，挺好吃的。然后各种蛋炒饭的话我大一大二的时候有一段时间吃过，但不是很好吃。有点腻。
1.2 一楼的淮南牛肉汤。这家店的话，店主人有点抠门，接待让人感觉怪怪的。我在他家一般就吃淮南牛肉汤10r，必须加特别的辣椒（牛油辣椒），不然没味道。还有就晚上当宵夜的鸡蛋米线6r，之前他家有自家腌制的辣椒可以免费加，现在他去掉了。太抠门了。然后这个鸡蛋米线也失去了灵魂，我很少吃了。就是今年才撤下的，以后加上了推荐去吃。但估计不会加了。刚开业的时候他家还有免费的榨菜，很好吃也早就提供了。我说开门做生意何必呢？要不就干脆一开始就不上，上了然后撤下真的难受。
1.3 那个角落的水果捞。一楼and二楼角落风水一直不好，印象中倒闭了好多次。赌一把这家继续倒闭。其实之前一楼两家都不错。但都倒闭了，我觉得水果捞这家完全没有之前两家好啊。
1.4 一楼的山东杂粮煎饼and炒饭。炒饭多，但不好吃。这些加的料真的不好吃，而且吃炒饭没有汤失去灵魂。不推荐。杂粮煎饼好吃，但是找不到时间吃。早上赶根本没有时间，中午晚上都吃不饱。
1.5 一楼的快餐两家。学士缘的阿姨真的难受，估计他们都没有孩子的吧，那一点点饭哪个成年男生吃得饱？（我得反思一下是不是因为我饭量大的原因）让他加饭，他也才加一点点，大家都不容易何必为难呢。但是菜有时候还是很好吃的，而且量也足，价格也便宜。早些时候甚至有爆炒鱿鱼这道菜，非常喜欢。其他的一些菜也挺好吃的，只是那两个阿姨以走过去就问“在这吃吗”，可我只是单纯想看看什么菜鸭。。。。非学士缘那家饭量足，菜不好吃。一般不太去吃。
1.6 一楼麻辣香锅。贵但好吃。偶尔会吃。推荐加豆腐果就够了，之前荤菜有冰冻鱿鱼，现在没了。也没啥好吃的荤菜，还是豆腐果加面饼就基本饱了。
1.7 一楼的饺子。等的时间巨巨巨巨巨长。。。。配料不足，不是很好吃。加上有一次我明明刷了卡了但她说我没刷让我再刷一次。幸好有校园卡有消费记录。从那次我就没有在去吃过。他是自包有一种怪怪的味道（好像是蘸水的那个泡沫味道，很难闻）。让他倒闭吧。
1.8 一楼韩式。等的时间长长长。豆腐汤啥玩意，真豆腐汤。推荐真的不想吃饱的时候去吃。我觉得韩式可能就是吃不饱的代名词吧。其他的泡菜啥的也不好吃。个人口味吧。
1.9 一楼的拉面。和西区的好像一样。我一般肚子有点饱的时候会去吃，鲜笋拌面11r或者拌面+牛肉汤12r。但之前舍友和我说他家的拌面有一股骚味，让我每次吃都感觉确实有。就没去了。
1.10 一楼角落。怎么说呢？这家好像很多人推荐的。我个人的话其实觉得没啥好吃的，一般黄焖鸡12r没有楼上14r好吃，之前他家肉多优势，现在也没了。烤肉饭的话总让我感觉有肥肉杂在里面，我吃不来肥肉所以有点难受。其他的吃过觉得不好吃，我对于又甜又咸的完全不适应。
1.11 一进门的两家是早餐和晚餐供应的。大一的时候吃。赶时间买个包子路上吃挺好的。就是如果肉包不新鲜的话就非常的难吃。我是完全吃的出来。那种不新鲜的感觉让人感到呕吐。就为了不遇到这种不新鲜的包子我就不怎么去吃。他家新鲜的肉包就很好吃。

以上就是一楼的主食部分，一楼的两家店的话。
2.1 轰炸鸡排 我四年没吃过他家鸡排。有一段时间口袋里面有钱，整天早上吃他家的各种咖啡。是他自己调制的，矮矮的大叔调的不行，太甜了。高一点的还行，但也有点甜。不过好喝是挺好喝的，也提神。喝一杯去数理化图书馆绝对能干一上午。但是就是和咖啡有一个明显的副作用，增进排泄。我也是喝了他家咖啡才意识到的。
2.2 另一家的话就是面包店。我其实有想面包店打工的念头的。还是比较喜欢吃面包的。可是面包真的贵。我记得以前的话一袋面包10r是一大袋了，这家的话一袋才6-7小片。太贵了。这两家店都是提供饮料的。肥宅快乐水当然是我的首选啦啦啦。3.5r和4.5r的两种包装。其中之前有小罐装的2.5r挺不错的，然后被学校撤下了。另外，值得注意的是，大瓶的矿泉水有三种牌子，农夫山泉and纯悦andxx，后两种中我总觉得有一种喝了容易拉肚子。有点问题。运动完了有时候就要喝大的才带劲。

二楼的美食的话
3.1 麻辣烫。麻辣烫让我们自己加辣椒麻油（而且就这两种）真的敷衍。我大二的时候吃过几次，辣椒也不是特别香，感觉不是很好吃。
3.2 随园山庄快餐。四年几乎就去过一两次。在绝对的优势U选上面，这个快餐是真的他不起头来。而且那个点餐台设置的那么高，什么意思啦。是不让我们点餐的意思呗？有想法没钱财莫点菜？
3.3 黄焖鸡。好吃。但之前是一个小哥哥，现在变成一个小姐姐。总感觉失去了一些味道。而且也变少了。但我还是要说一句，不加中辣没有灵魂。
3.4 麻辣香锅。便宜，味道还行。比起一楼的便宜太多了，而且荤素统一价格。就是吃完有一股嘴里总是有一股味道挥之不去。还行吧。建议人少点的时候去吃，自在。
3.5 新来的一家韩式之类的，没去吃过。舍友封杀了他家。
3.6 面。这个面真的简单，开水一烫就出锅，加点菜就加钱。也没啥味道。简直侮辱小面。
3.7 角落的新来的牛肉汤？有牛肉？？？再见
3.8 U选的韩式料理。包含炒饭，杂拌以及其他一些乱七八糟的东西。我感觉小姐姐们可能喜欢点。我不怎么喜欢。
3.9 U选蛋包饭。太太太咸咸咸了。太干了。之前那个胖大叔咋走了呢。他都认识我了。虽然不好吃，但偶尔吃吃还行吧。还有水果。那个番茄酱有时候追求变态的感觉的话可以加，其他时候加了就感觉有点变态，又甜又咸的完美诠释。
3.10 U选。强烈推荐早餐U选的白粥+包子（三角香菇包>肉包>。。。）。晚餐偶尔吃吃那个麻辣鸡就挺好吃的，还下饭。晚餐饭有点少可以加个红糖馒头好吃。中午我一般不吃U选。
3.11 鸡汤面。吃了四年的鸡汤面也是时候说好再见了。阿姨也真的认识我了，有一段时间每天早上都吃云吞面，然后去了之后直接刷卡。都不需要交流。然后这学期突然解锁了鸡汤米线，真的好吃。但因为我的一句太辣了，阿姨加辣椒就总是小心翼翼的。其实早上的话太辣是吃不消，但晚上的时候追求刺激，还是辣一点好。这家我有机会一定会再来吃的。毕竟都这么熟了。两位阿姨人都很好。不过我喜欢话少一点的。

二楼的三家商店，我只去过一家。还是买的咖啡。二楼的美式咖啡，让人绝望的苦味。
西区的一些店：
4.1 一楼的拉面的话和东区应该是一家的，味道也是一样的味道。只能说还行。
4.2 一楼的牛杂米线。有一段时间很喜欢+2块钱饼。但后来突然意识到，这饼也太难咬了吧。而且这个牛杂汤真的是喝汤啊，完全吃不饱啊。
4.3 角落的瓦香鸡。好吃就是有点少，也就鸡块鸡肉，让他撒点生辣椒会香很多
4.4 二楼馄饨。非常nice，他家的辣椒非常好吃（辣）。可以把6r的小馄饨命名为辣油馄饨。非常棒，但是得找一个身体状态好的时候去吃。他家其他的馄饨，饺子什么的都一般。那个牛肉汤的话还不如东区的，就是有那个辣油加成就变强了。
4.5 二楼角落的面。那个面至少还是现烧的，学校很少面能做到现烧，这样汤就入味些，更加好吃。但是他加那些味精，盐都是靠随手来的，有时候控制的真的不好。那个网红泡面系列我感觉是在搞笑的。我一般点八鲜面（还是七鲜面），还有老卤面。里面有一个鸡蛋非常不错，增加别样的感觉。

西区的话一般都是抽空去吃的，吃的不多，也就这几家去的多点。
